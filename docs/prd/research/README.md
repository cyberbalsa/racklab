# PRD Research Notes

This folder contains research notes that support the RackLab PRD. The files mirror the main PRD sections and record the source-backed details that informed each requirement.

The goal is not to replace the PRD. The goal is to keep implementation-relevant research close to the product requirements without making any single PRD file too large.
